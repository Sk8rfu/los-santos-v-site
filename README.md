# Los Santos V - Archived Client Concept Website (2021–2022)

This repository contains an archived concept website for **Los Santos V**, an early idea for a custom multiplayer client inspired by GTA V.  
The original vision was to create a standalone client that could one day compete with platforms like **FiveM**.

The project was started in **2021** and completed in **2022**, during a time when I was still learning the basics of web development.  
Because I didn’t yet have the knowledge to build a real multiplayer client or backend, many of the links in the website were replaced with external references such as:

- **Rockstar Games**
- **FiveM**
- **Epic Games**
- **Steam**

These were used as placeholders to represent the idea of a future Los Santos V client.  
The project is preserved exactly as it was back then - as a snapshot of my early development journey and the original concept.

---

## 📝 Project Background

This project was created during my early learning phase.  
At the time, I didn’t have the technical knowledge to build:

- a real multiplayer client  
- networking systems  
- backend services  
- game integration  

Because of that, the site uses placeholder links to GTA‑related pages such as Rockstar Games, FiveM, Epic Games, and Steam.  
These links symbolized the direction the project was meant to take.

Today, the project is open‑sourced for nostalgia, transparency, and documentation of my progress as a developer.

## 📂 Project Structure

The website consists of several static HTML pages styled with external CSS files:

### **HTML Pages**
- `home.html` - Main landing page  
- `download.html` - Download section with platform links  
- `pictures.html` - Gallery of Los Santos images  
- `contact.html` - Old placeholder support page  
- `about.html` - About the fictional city and project  

### **CSS Files**
- `style.css` - Home page styling  
- `style2.css` - Pictures page styling  
- `style3.css` - Download page styling  
- `style4.css` - Contact page styling  
- `style5.css` - About page styling  

### **Assets**
- `logo2.png` - Main logo  
- `1.jpg` … `9.png` - Gallery images  
- Additional background images depending on CSS  

---

## 🎯 Project Purpose

The goal of Los Santos V was to experiment with:

- Designing a standalone multiplayer client concept  
- Exploring UI/UX ideas for a GTA‑style launcher  
- Creating a website that could serve as the hub for the project  
- Learning HTML, CSS, and basic web structure  

Although the client itself was never completed, the website remains as a preserved concept.
